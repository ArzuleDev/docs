## Coming Soon

LangChain integration will provide:

- Chain execution tracking with inputs and outputs
- LLM call instrumentation with token counts
- Tool invocation monitoring
- Agent action and decision tracking
- Retriever operation capture
- Agent handoff detection for multi-agent workflows
- LangGraph support for graph-based workflows
