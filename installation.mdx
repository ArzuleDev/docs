## Requirements

- Python 3.10 or higher
- pip, poetry, or uv package manager
- A virtual environment (recommended)

## Install the SDK

The SDK has a lightweight core with optional framework integrations:

### Core Installation

The core package includes **Claude Code integration** and works on all supported Python versions:

<Tabs>
<Tab title="pip">
```bash
pip install arzule-ingest
```
</Tab>

<Tab title="poetry">
```bash
poetry add arzule-ingest
```
</Tab>

<Tab title="uv">
```bash
uv add arzule-ingest
```
</Tab>
</Tabs>

### With Framework Integrations

For CrewAI, LangChain, or AutoGen support, install with extras:

<Tabs>
<Tab title="pip">
```bash
# Single framework
pip install arzule-ingest[crewai]
pip install arzule-ingest[langchain]
pip install arzule-ingest[autogen]

# Multiple frameworks
pip install arzule-ingest[crewai,langchain]

# All frameworks
pip install arzule-ingest[all]
```
</Tab>

<Tab title="poetry">
```bash
poetry add "arzule-ingest[crewai]"
poetry add "arzule-ingest[all]"
```
</Tab>

<Tab title="uv">
```bash
uv add "arzule-ingest[crewai]"
uv add "arzule-ingest[all]"
```
</Tab>
</Tabs>

<Info>
Framework extras require Python versions supported by each framework. CrewAI currently requires Python &lt;3.14. The core SDK (including Claude Code) works on Python 3.10-3.14.
</Info>

## Configuration

### Environment variables

Set your Arzule credentials as environment variables:

```bash
export ARZULE_API_KEY="your-api-key"
export ARZULE_TENANT_ID="your-tenant-id"
export ARZULE_PROJECT_ID="your-project-id"
```

<Tip>
You can find your credentials in the [Arzule dashboard](https://app.arzule.com) under Settings > API Keys.
</Tip>

### Programmatic configuration

You can also pass credentials directly to `init()`:

```python
import arzule_ingest

arzule_ingest.init(
    api_key="your-api-key",
    tenant_id="your-tenant-id",
    project_id="your-project-id",
)
```

<Warning>
Avoid hardcoding credentials in source code. Use environment variables or a secrets manager in production.
</Warning>

## Verify installation

Test that the SDK is installed correctly:

```python
import arzule_ingest

# Should print the version number
print(arzule_ingest.__version__)
```

## What's included

### Core Package
- **Claude Code** - Hook-based instrumentation for Anthropic's Claude Code
- **SOC2 compliance** - Encryption at rest for trace files
- **CLI tools** - `arzule-claude`, `arzule-claude-install`, trace viewer

### Optional Extras
- **`[crewai]`** - Automatic instrumentation for CrewAI agents
- **`[langchain]`** - Callback-based tracing for LangChain/LangGraph
- **`[autogen]`** - Automatic instrumentation for Microsoft AutoGen
- **`[all]`** - All framework integrations
